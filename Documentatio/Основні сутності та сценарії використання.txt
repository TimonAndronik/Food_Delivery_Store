Сутності:
User (користувач) → ім’я, email, пароль, тип (фізична особа / бізнес).
Product (товар) → назва, категорія, ціна, кількість на складі.
Order (замовлення) → користувач, список товарів, статус.
Delivery (доставка) → адреса, дата.
Cart (кошик) → тимчасове збереження обраних товарів.

Основні сценарії використання:
Користувач заходить на сайт та переглядає каталог товарів.
Додає продукти в кошик.
Оформляє замовлення (обирає доставку, спосіб оплати).
Замовлення передається кур'єру.

Етапи реалізації проекту:
Етап 1 → Статична лендінг-сторінка (html + css).
Етап 2 → Простий бекенд (Python Flask), база даних, API для товарів.
Етап 3 → Реалізація кошика та замовлень.
Етап 4 → Авторизація користувачів, розмежування ролей (User/Admin).

Етап 5 → Інтеграція платежів і доставки.
